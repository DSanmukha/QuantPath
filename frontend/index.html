How It Works

Three steps from raw data to actionable insights

1

Select a Stock

Enter an Indian BSE/NSE stock ticker (e.g., RELIANCE.BSE) and fetch real historical price data via the Alpha Vantage API.

2

Run Simulation

Drift and volatility are auto-calculated from historical data. The engine generates hundreds of GBM price paths using proper Box-Muller random sampling.

3

Analyze Results

Review price path charts, distribution histograms, risk metrics (VaR, CI), compare stocks, and export CSV reports.
